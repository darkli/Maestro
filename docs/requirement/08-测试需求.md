## 8.5 测试策略建议

### 8.5.1 Mock 策略

| 外部依赖 | Mock 方式 |
|---------|----------|
| LLM API 调用 | Mock openai/anthropic 客户端，返回预设 JSON |
| Claude Code CLI | Mock subprocess.run，返回预设 RunResult |
| Telegram API | Mock python-telegram-bot 库 |
| 文件系统 | 使用 pytest tmp_path fixture |

### 8.5.2 测试数据

建议维护以下测试 fixture：

| Fixture | 内容 |
|---------|------|
| sample_state | 各状态的 state.json 样本 |
| sample_checkpoint | 恢复用的 checkpoint.json |
| sample_actions | 五种 action 类型的 JSON 样本 |
| sample_tool_output | Claude/Generic 模式的输出样本 |
| sample_config | 各种配置组合 |
