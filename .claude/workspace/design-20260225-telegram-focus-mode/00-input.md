# 原始需求输入

**功能名称**: telegram-focus-mode
**提交时间**: 2026-02-25 16:50
**用户描述**:

Telegram Bot 任务关注模式。用户通过 bot 发送任务后，bot 告知任务开始执行并自动关注该任务。关注期间，只有 Claude Code 有输出时才把输出内容返回给用户，否则只提示 "Claude Code vibing..." 之类的消息。多任务并行时一次只能关注一条任务，新发起的任务自动关注。用户可通过新增指令切换关注的任务。

**核心诉求**:

1. 当前每轮 Turn 通知只有数字（费用/耗时），没有实际内容反馈
2. 需要看到 Claude Code 的实际输出和 Manager 的决策内容
3. 多任务场景下只关注一个任务，避免信息洪泛
4. 新增 /focus 命令切换关注

**附加上下文**:

当前 Orchestrator 的 `_telegram_push_turn` 只推送 turn 序号、费用、耗时三个数字，不包含编码工具输出和 Manager 决策内容。用户无法通过 Telegram 了解任务实际进展。
